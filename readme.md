## HCL Hackathon - Simple Medical Portal

# Problem:

To create a solution for a healthcare wellness and preventive care portal that seamlessly integrates front-end and back-end.

- **Responsive Design**: Mobile-first layout using CSS for adaptability across devices.
- **Authentication**: Mocked authentication system with localStorage for persistence.
- **Role-Based Access Control**: Separate access for Patients and Doctors (Healthcare Providers).
- **Profile Management**: Editable profile pages for users.
- **Compliance with healthcare privacy standards**:
- **Dashboards**: Dedicated dashboards for Patients and Doctors to manage interactions.

The portal should provide usability and key preventive care features.

# Emphasis: 
MVP that demonstrates key aspects of problem statement, with emphasis on security, personalization and healthcare compliance.
